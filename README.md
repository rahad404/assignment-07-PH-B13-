# KEEN-KEEPER

**ProgrammingHero B-13 — Assignment 7**

A modern friendship tracker built with **React** and **Next.js**. Keen Keeper helps you view friend details, log interactions, and visualize relationship activity.

## Live Demo
- https://keen-keeper-one-chi.vercel.app/

## GitHub
- https://github.com/rahad404/assignment-07-PH-B13-

## Key Features
- View detailed friend profiles
- Log interactions as **Call**, **Text**, or **Video**
- Display interaction distribution with a **pie chart**

## Built With
- **Next.js**
- **React**
- **Tailwind CSS**
- **DaisyUI**
- **Recharts**
- **Lucide React** icons
- **React Hot Toast**

## What I Added / Used
- Clean, responsive UI with **Tailwind CSS**
- Reusable components for friend cards, detail pages, timeline, and stats
- `daisyUI` styling utilities for buttons, cards, and layout polish
- `recharts` for interactive chart visualization
- `lucide-react` for lightweight iconography
- Toast feedback for interaction logging

## Project Structure
- `src/app/` — app routes and page layouts
- `src/components/` — reusable UI components
- `src/context/` — timeline state management
- `src/data/` — friend data source

## Notes
This assignment focuses on building a polished friend management experience with real-time interaction logging and data visualization.
